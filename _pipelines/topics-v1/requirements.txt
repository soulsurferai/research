## Installation


## Notes
- All embeddings are pre-computed in Qdrant (no need for sentence-transformers)
- UMAP provides better dimensionality reduction than PCA for topic discovery
- HDBSCAN with smart parameters handles varying community sizes
- Matplotlib backend issues may require: `export MPLBACKEND=Agg`

## Verified Working Versions
- Python 3.11 (via conda environment core311)
- macOS compatible (tested on M1 MacBook Air)
- Memory efficient for 5000+ document analysis
